# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0]

### Added

- Initial release of Reth Helm chart
- Support for Reth Ethereum execution client
- JWT authentication for Engine API
- Configurable RPC, WebSocket, and Engine API endpoints
- Metrics and Prometheus ServiceMonitor support
- Grafana dashboard integration
- Custom reth.toml configuration support
- P2P networking with optional NodePort exposure
- StatefulSet deployment with persistent storage
- RBAC and ServiceAccount configuration
